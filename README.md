# swdb_2017_tools

A collaborative Python package built by participants of the Summer Workshop on the Dynamic Brain

## Usage

```Python
import swdb_2017_tools
```

## Installation

```
git clone https://github.com/AllenInstitute/swdb_2017_tools.git
cd swdb_2017_tools
pip install -e ./
```

### Requirements

- allensdk
- numpy
- matplotlib
- pandas
